= Deploying the eventrouter in {product-title}

You can configure {product-title} to use the *eventrouter* pod to collect Kubernetes events, which are forwarded to Fluentd and Elasticsarch. 

.Procedure

To deploy the eventrouter:

.  Edit the Ansible inventory file, by default *_/etc/Ansible/hosts_*.

. Add or set the `openshift_logging_use_ops` to true.
+
----
openshift_logging_install_eventrouter=true
----

. Specify other parameters as needed:
+
include::modules/efk-logging-deploy-variables.adoc[tag=eventrouter-vars]

. Run the Ansible playbook as shown below.

= Creating a quota

You can create a quota to constrain resource usage in a given project.

.Procedure

. Define the quota in a file. See
xref:../applications/quotas/quotas-setting-per-project.adoc#quotas-sample-resource-quota-definitions[Sample resource quota definitions]
for examples.

. Use the file to create the quota and apply it to a project:
+
----
$ oc create -f <file> [-n <project_name>]
----
+
For example:
+
----
$ oc create -f core-object-counts.yaml -n demoproject
----

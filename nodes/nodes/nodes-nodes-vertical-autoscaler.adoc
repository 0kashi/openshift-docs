The {product-title} Vertical Pod Autoscaler Operator (VPA) frees you from needing to set up-to-date 
resource limits and requests for the containers in their pods. The VPA Recommender automactically
computes the recommended resource requests for pods based on historical and current usage of the resources. 

The Vertical Pod Autoscaler is currently a Technology Preview feature.
